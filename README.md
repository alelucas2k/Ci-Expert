# Ci-Expert

Este repositório centraliza todas as atividades, práticas de laboratório e projetos desenvolvidos durante a residência

## Estrutura do Repositório

O projeto está organizado da seguinte forma:

* **`worskpace/`**: Contém os diretórios de trabalho onde os projetos e laboratórios são desenvolvidos (ex: labs de circuitos, design digital).
* **`tcl_test/`**: Pasta dedicada a scripts de automação e testes utilizando a linguagem **TCL** (Tool Command Language), comum em ferramentas de EDA.

## Tecnologias e Ferramentas
- **Linguagens:** SystemVerilog, TCL, Shell Script.
- **Foco:** Verificação de Hardware e Automação de Fluxo EDA.
