# Laboratório S1 - Workspace

Este diretório contém os projetos e práticas referentes à etapa **S1**. A estrutura foi gerada automaticamente utilizando o script de esqueleto de projetos fornecido no material de apoio.

## Organização Interna

- **`prj/`**: Contém as pastas de projetos específicos (ex: `calc`, `first_rtl`).
    - Cada projeto possui subpastas para `rtl` (código fonte), `tb` (testbenches) e `docs` (documentação específica).
- **`scripts/`**: Scripts úteis para o desenvolvimento desta etapa, incluindo automação de Git e geração de novos projetos.
