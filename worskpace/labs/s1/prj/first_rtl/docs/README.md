# Projeto: First RTL (Porta AND)

Este é o projeto inicial da residência, focado na configuração do ambiente de simulação e na implementação de uma porta lógica básica. Serve como o "Hello World" do fluxo de design digital.

## Conteúdo do Projeto

* **`rtl/`**: Contém o módulo `dut.sv` (Design Under Test).
* **`tb/`**: Contém duas versões de testbench:
    * `tb_top.sv`: Teste inicial de fluxo e repetição.
    * `tb_top_v2.sv`: Verificação funcional completa do hardware.
